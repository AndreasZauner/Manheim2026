# Lehrgrabung Kerpen-Manheim 2026 – GitHub-fertiges Projekt

Dieses Paket ist dafür vorbereitet, als **GitHub-Repository** hochgeladen und danach mit **Cloudflare Pages** und **Supabase** verbunden zu werden.

## Was ist in diesem Paket?

- `index.html` – die Web-App
- `styles.css` – Gestaltung
- `app.js` – Logik der App
- `config.js` – hier tragen Sie Ihre Supabase-Daten ein
- `config.example.js` – Vorlage
- `supabase/schema.sql` – Datenbankstruktur
- `supabase/seed.sql` – Startdaten
- `ANLEITUNG_EINFACH.md` – sehr einfache Schritt-für-Schritt-Anleitung
- `CHECKLISTE_START.txt` – Kurzcheckliste
- `.gitignore` – für GitHub vorbereitete Ignorierliste

## Empfohlener Weg

1. Projekt bei **GitHub** anlegen
2. diese Dateien hochladen
3. Projekt bei **Supabase** anlegen
4. `schema.sql` und `seed.sql` ausführen
5. `config.js` mit den Supabase-Daten füllen
6. Projekt mit **Cloudflare Pages** verbinden
7. Seite online testen

Die genaue einfache Anleitung steht in `ANLEITUNG_EINFACH.md`.
